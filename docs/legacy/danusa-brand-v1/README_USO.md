# COMO USAR ESTE PACOTE

## Uso no Claude
1. Anexe o arquivo danusa_master.md.
2. Escreva sua tarefa.
3. Quando precisar de maior controle, anexe também branding.md, rules.md, images.md e prompts.md.

## Frase curta para usar no Claude
Use os arquivos anexados como sistema fixo da marca Danusa. Não altere a paleta Obsidian Clinical e não desvie do posicionamento.

## Arquivos
branding.md: branding book completo.
rules.md: regras operacionais.
images.md: sistema visual e prompts de imagens.
prompts.md: prompts prontos para site, posts e imagens.
danusa_master.md: arquivo único para uso rápido.
